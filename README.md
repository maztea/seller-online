# Seller Online Boilerplate

A premium, full-stack boilerplate built with the modern web stack. This project is designed for speed, security, and scalability.

## 🚀 Tech Stack

- **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/)
- **Authentication:** [Better Auth](https://better-auth.com/)
- **ORM:** [Drizzle ORM](https://orm.drizzle.team/)
- **Database:** [PostgreSQL](https://www.postgresql.org/)
- **Caching:** [Redis](https://redis.io/)
- **Containerization:** [Docker](https://www.docker.com/) & [Docker Compose](https://docs.docker.com/compose/)
- **Task Runner:** [Makefile](https://www.gnu.org/software/make/)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v20+)
- Docker & Docker Compose
- `make` (optional, but recommended)

### Environment Variables

Copy `.env.example` to `.env` (already initialized in this boilerplate):

```properties
DATABASE_URL=postgresql://user:password@localhost:5432/seller_online
BETTER_AUTH_SECRET=p8H5KxR4mN2qL9vW1zY3cX0bJ7aE6S4D
BETTER_AUTH_URL=http://localhost:3000
REDIS_URL=redis://localhost:6379
```

### Local Development

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

### Docker Deployment

To spin up the entire stack (App, DB, Redis) with one command:

```bash
make deploy
```

## 📦 Makefile Commands

| Command | Description |
| :--- | :--- |
| `make build` | Build the Next.js app and Docker images |
| `make deploy` | Start all services in detached mode |
| `make migrate` | Push database schema changes via Drizzle |
| `make seed` | Seed the database with initial data |
| `make clean` | Stop containers, remove volumes, and clean build artifacts |

## 🏗️ Project Structure

- `src/app`: Next.js App Router (pages and API routes)
- `src/components`: UI components (including shadcn/ui)
- `src/db`: Drizzle schema and migration files
- `src/lib`: Shared utilities (auth clients, db initialization, redis)
- `public`: Static assets
